## 📋 Sobre

Portfolio profissional para Alla Kayan, cantora e artista, com sistema de traducao de conteudo.

## 🚀 Tecnologias

| Tecnologia | Versao | Uso |
|------------|--------|-----|
| React | 19.x | UI Library |
| Vite | 6.x | Build Tool |
| TypeScript | 5.x | Linguagem |
| TailwindCSS | 4.x | Estilizacao |
| Express | - | Backend API |
| SQLite | - | Banco de dados |

## 🏃 Quick Start

```bash
# Clonar repositorio
git clone https://github.com/ricardo-camilo-programador-frontend-web/alla-kayan-portfolio2.0.git

# Instalar dependencias
pnpm install

# Rodar desenvolvimento
pnpm dev
```
